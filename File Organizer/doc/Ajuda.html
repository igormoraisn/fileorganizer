    • Sobre o programa

    O gerenciador de registros é um software de criação e manipulação de registros. Foi idealizado e implementado como trabalho da disciplina de Estrutura de Dados II, supervisionado pelo Prof. Dr. Tarcísio da Rocha.

    Características

    São apresentadas a seguir, algumas das principais características do ArqGer:

  • Escrito em Java, uma das linguagens mais populares do mundo.

  • Utiliza o Método de Brent.

  • Utiliza a.

  • Baseado no princípio minimalista.

    • Utilizando o ArqGer

    A interface do ArqGer dispõe de 3 opções:


  • Adicionar um novo registro.

  • Remover um registro.

  • Pesquisar um registro.

    • Adicionando registros

    Escolhendo a opção de adicionar um registro, um formulário irá ser retornado e nele, devem ser preenchidos todos os dados relativos ao registro, sendo eles: matrícula, nome do usuário, endereço, sexo, idade e email.

    Após preencher todos os dados, deve-se clicar no botão "enviar" para armazenar os dados do registro em um endereço válido.

    Removendo registros

    Para remover um arquivo, você deve inserir a matrícula desejada no campo apresentado. Após a remoção será apresentada uma mensagem de confirmação e os dados do registro serão apresentados na tela.

    Pesquisando um registro

    Ao escolher a opção de busca de registro, a matrícula deverá ser digitada no campo apresentado. Todos os dados do registro cuja matrícula foi digitada, serão mostrados na tela.
